# SkillSpark — Learn. Build. Launch.

A fully responsive promotional website for a tech education startup built using HTML5, CSS3, Bootstrap 5, and vanilla JavaScript.

---

## Live Demo

https://iabdulsattarktk.github.io/Skill-Spark/ 
---

## Pages

- **Home** — Hero section, feature cards, Why Choose Us, testimonials carousel, footer
- **Courses** — 6 course cards with price badges, hover effects, modal popups
- **About** — Mission, progress bars, founder profile, milestone timeline
- **Register** — Responsive form with validation, dropdowns, radio buttons, file upload
- **Contact** — Contact form, address info, embedded Google Map
- **About Developer** — About developer 

---

## Features

- Fully responsive — Mobile, Tablet, Desktop
- Bootstrap 5 grid system
- Dark mode toggle (remembers preference)
- Scroll to top button
- AOS scroll animations
- Anime style UI with Cyberpunk color theme
- Form validation (is-valid / is-invalid)
- Testimonials carousel
- Course detail modal popups
- Embedded Google Map
- Custom CSS with smooth hover transitions
